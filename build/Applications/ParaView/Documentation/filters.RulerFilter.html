Ruler (RulerFilter)

Create a line along the input to use as a ruler
This filter creates a line along the object and defaults its representation to showing a ruler along that line.
Property Description Default(s) Restrictions
Input

Accepts input of following types:

  • vtkDataObject
Axis Select which axis to put the ruler along 0

The value(s) is an enumeration of the following:

  • X Axis (0)
  • Y Axis (1)
  • Z Axis (2)
