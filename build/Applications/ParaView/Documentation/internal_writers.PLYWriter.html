PLYWriter (PLYWriter)

Write polygonal data in Stanford University PLY format.
Property Description Default(s) Restrictions
DummyInput

The dataset must contain a field array (any)

EnableColoring Specify whether color information should be saved in the PLY file. If enable, the data source's coloring state i.e. the array to color with and the transfer function from the active view will be used. 0

Accepts boolean values (0 or 1).
ColorArrayName The array that will be used for saving color information when Enable Coloring is ON.

The value must be field array name.
LookupTable Set the lookup table to use for scalar mapping.
