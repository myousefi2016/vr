Tecplot Table Reader (TecplotTableReaderCore)

The Tecplot table reader reads a .DAT file containing tabular data into a spreadsheet view.
Property Description Default(s) Restrictions
FileName The name of the file to read.

The value(s) must be a filename (or filenames).
FileNameInfo
Max. Number of Records The maximum number of records to read. Use this to preview a file. 0
Number of header lines The number of lines that form the header. 2
Line with column names The line that contains the column names. 1
Skip #column name fields Number of fiels to skip while reading column names. 1
