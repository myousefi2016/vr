Bounded Plane (BoundedPlaneSource)

Property Description Default(s) Restrictions
BoundingBox Set the bounding box for the domain 0 1 0 1 0 1
Resolution This property specifies the number of divisions along the X axis of the parallelogram. 100
Center Specify the center of the plane. Works in conjunction with the plane normal to position the plane. Don't use this method to define the plane. Instead, use it to move the plane to a new center point. 0 0 0

The value must lie within the bounding box of the dataset. It will default to the mid in each dimension.

Normal Specify the plane normal. Works in conjunction with the plane center to orient the plane. Don't use this method to define the plane. Instead, use it to rotate the plane around the current center point. 0 0 1
