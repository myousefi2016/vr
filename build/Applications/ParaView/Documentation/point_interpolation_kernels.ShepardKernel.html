ShepardKernel (ShepardKernel)

This is an interpolation kernel that uses the method of Shepard to perform interpolation. The weights are computed as 1/r^p, where r is the distance to a neighbor point within the kernal radius R; and p (the power parameter) is a positive exponent (typically p=2).
Property Description Default(s) Restrictions
PowerParameter Specify the power parameter p. By default p=2. Values (which must be a positive, real value) != 2 may affect performance significantly. 2.0
