Append Reduce (AppendReductionFilter)

Property Description Default(s) Restrictions
Input Set the input to the Reduction filter.

Accepts input of following types:

  • vtkDataSet
PostGatherHelperName Set the algorithm that takes multiple inputs and produces a single reduced output. vtkAppendFilter
ReductionMode Set the reduction mode. Reducing all data to one processor means that the destination process will have all data while each other process will still have it's own data. Moving all to one processor means that the destination process will have all data while other process have no data anymore. Reduce all data to all processors is self explanatory. 2

The value(s) is an enumeration of the following:

  • Reduce all data to one processor (0)
  • Move all data to one processor (1)
  • Reduce all data to all processors (2)
ReduceTo Set the process to reduce to. 0
OnlyFrom If set to a non-negative value, then produce results using only the node Id specified. -1
GenerateProcessIds If true, the filter will generate vtkOriginalProcessIds arrays indicating the process id on which the cell/point was generated. 0

Accepts boolean values (0 or 1).
