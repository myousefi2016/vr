Force Time (ForceTime)

Filter used to explicitly request a specific time from the pipeline.
Property Description Default(s) Restrictions
Input This property specifies the input to the ForceTime filter.

Accepts input of following types:

  • vtkDataSet
IgnorePipelineTime If set to 0, this filter will do nothing, only shallow copy the input to the output. If set to 1, this filter will always request the ForcedTime to the pipeline, ignoring time requests. 1

Accepts boolean values (0 or 1).
ForcedTime This property specifies the time to request. If the IgnorePipelineTime property is set, then this value will override any time request in the VTK pipeline. 0.0
