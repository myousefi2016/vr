EllipsoidalGaussianKernel (EllipsoidalGaussianKernel)

This is an interpolation kernel that returns the weights for all points found in the ellipsoid defined by radius R in combination with local data (normals and/or scalars).
Property Description Default(s) Restrictions
UseNormals Specify whether vector values should be used to affect the shape of the Gaussian distribution. 1

Accepts boolean values (0 or 1).
UseScalars Specify whether scalar values should be used to scale the weights. 0

Accepts boolean values (0 or 1).
Radius Specify the radius of the kernel. Points within this radius will be used for interpolation. If no point is found, then the closest point will be used. 1.0
Sharpness Specify the sharpness (i.e., falloff) of the Gaussian. By default Sharpness=2. As the sharpness increases the effects of distant points are reduced. 2.0
Eccentricity Specify the eccentricity of the ellipsoidal Gaussian. A value=1.0 produces a spherical distribution. Values less than 1 produce a needle like distribution (in the direction of the normal); values greater than 1 produce a pancake like distribution (orthogonal to the normal). 2.0
