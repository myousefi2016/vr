Environment Annotation (EnvironmentAnnotation)

Allows annotation of user name, date/time, OS, and possibly filename.
Apply to any source. Gui allows manual selection of desired annotation options. If the source is a file, can display the filename.
Property Description Default(s) Restrictions
Input Set the input of the filter.

Accepts input of following types:

  • vtkDataObject
DisplayUserName Toggle User Name Visibility. 1

Accepts boolean values (0 or 1).
DisplaySystemName Toggle System Name Visibility. 1

Accepts boolean values (0 or 1).
DisplayDate Toggle Date/Time Visibility. 1

Accepts boolean values (0 or 1).
DisplayFileName Toggle File Name Visibility. 1

Accepts boolean values (0 or 1).
DisplayFilePath Toggle Show Full File Path. 1

Accepts boolean values (0 or 1).
FileName Annotation of file name.
