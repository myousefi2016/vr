GeneralizedKernelBase (GeneralizedKernelBase)

Property Description Default(s) Restrictions
KernelFootprint Specify the interpolation basis style. By default, a Radius style is used (i.e., the basis is defined from all points within a specified radius). However, it is also possible to select the N closest points (N Closest). Note that in most formulations the Radius style is assumed as it provides better mathematical properties. However, for convenience some bases are easier to use when the N closest points are taken. 0

The value(s) is an enumeration of the following:

  • Radius (0)
  • N Closest (1)
Radius If the interpolation basis style is Radius, then this method specifies the radius within which the basis points must lie. 1.0
NumberOfPoints If the interpolation basis style is N Closest, then this method specifies the number of the closest points used to form the interpolation basis. 8
