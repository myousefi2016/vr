Point Plane Interpolator (PointPlaneInterpolator)

This probes a point cloud `PC` on the Input with a set of points `P` (the filter Source), interpolating the data values from `PC` onto `P`. Note however that the descriptive phrase "point cloud" is a misnomer: PC can be represented by any dataset type, with the points of the dataset forming PC. Similary, the output P can also be represented by any vtkDataSet type; and the topology/geometry structure of P is passed through to the output along with the newly interpolated arrays.
Property Description Default(s) Restrictions
Source The data source that provides the structure to be interpolated on.

Accepts input of following types:

  • vtkDataSet

The value can be one of the following:

  • BoundedPlaneSource (extended_sources)
