Point Interpolator (PointInterpolatorBase)

This probes a point cloud `PC` on the Input with a set of points `P` (the filter Source), interpolating the data values from `PC` onto `P`. Note however that the descriptive phrase "point cloud" is a misnomer: PC can be represented by any dataset type, with the points of the dataset forming PC. Similary, the output P can also be represented by any vtkDataSet type; and the topology/geometry structure of P is passed through to the output along with the newly interpolated arrays.

Kernel Specify an interpolation kernel. By default a vtkVoronoiKernel is used (i.e., closest point). The interpolation kernel changes the basis of the interpolation.

The value can be one of the following:

  • VoronoiKernel (point_interpolation_kernels)
  • EllipsoidalGaussianKernel (point_interpolation_kernels)
  • GaussianKernel (point_interpolation_kernels)
  • LinearKernel (point_interpolation_kernels)
  • ProbabilisticVoronoiKernel (point_interpolation_kernels)
  • ShepardKernel (point_interpolation_kernels)
  • VoronoiKernel (point_interpolation_kernels)

NullPointsStrategy Specify a strategy to use when encountering a "null" point during the interpolation process. Null points occur when the local neighborhood (of nearby points to interpolate from) is empty. If the strategy is set to MaskPoints, then an output array is created that marks points as being valid (=1) or null (invalid =0) (and the NullValue is set as well). If the strategy is set to NullValue, then the output data value(s) are set to the NullPoint value (specified in the output point data). Finally, the default strategy ClosestPoint is to simply use the closest point to perform the interpolation. 1

The value(s) is an enumeration of the following:

  • Mask Points (0)
  • Null Value (1)
  • Closet Point (1)
ValidPointsMaskArrayName If the NullPointsStrategy is "Mask Points", then an array is generated for each input point. This vtkCharArray is placed into the output of the filter, with a non-zero value for a valid point, and zero otherwise. The name of this masking array is specified here. vtkValidPointMask
NullValue Specify the null point value. When a null point is encountered then all components of each null tuple are set to this value. By default the null value is set to zero. 0.0
