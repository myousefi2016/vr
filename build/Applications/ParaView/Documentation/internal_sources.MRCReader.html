MRC Reader (MRCReader)

Reads a MRC file into an image data.
The MRC reader reads MRC (Medical Research Council) image files, and the output is a uniform rectilinear (image/volume) dataset. The default file extension is .mrc but .st, .rec and .ali extensions are known to be commonly used as well.
Property Description Default(s) Restrictions
FileName The name of the file to load.

The value(s) must be a filename (or filenames).
