NetCDF CAM reader (NetCDFCAMReaderCore)

Core for NetCDF CAM reader.
Property Description Default(s) Restrictions
FileName This property specifies the file name to read.

The value(s) must be a filename (or filenames).
ConnectivityFileName This property specifies the file name to read for the grid connectivity.

The value(s) must be a filename (or filenames).
PointsFileName This property specifies the file name to read for the points and field information.

The value(s) must be a filename (or filenames).
VerticalDimension Specify whether to read variables from the single layer, from midpoint layers or from interface layers. 1

The value(s) is an enumeration of the following:

  • Single layer [time, ncol] (0)
  • Midpoint layers [time, lev, ncol] (1)
  • Interface layers [time, ilev, ncol] (2)
PointArrayInfo
PointArrayStatus This property lists which NetCDF CAM point variables to load.

The list of array names is provided by the reader.
SingleMidpointLayer Load only one layer from variables on the midpoint layers. The layer is specified by MidpointLayerIndex. 0

Accepts boolean values (0 or 1).
MidpointLayersRangeInfo
MidpointLayerIndex Specifies which layer to load from the midpoint layers. Note that the layer indexes 0 to n-1 map to Z values n-1 to 0. 0
SingleInterfaceLayer Load only one layer from variables on the interface layers. The layer is specified by InterfaceLayerIndex. 0

Accepts boolean values (0 or 1).
InterfaceLayersRangeInfo
InterfaceLayerIndex Specifies which layer to load from the interface layers. Note that the layer indexes 0 to n-1 map to Z values n-1 to 0. 0
CellLayerRight DEPRECATED in ParaView 4.4. No longer needed/suported. 1

Accepts boolean values (0 or 1).
TimestepValues Available timestep values.
