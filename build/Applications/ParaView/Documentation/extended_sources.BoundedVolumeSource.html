Bounded Volume (BoundedVolumeSource)

Property Description Default(s) Restrictions
Origin Set the origin for the volume. 0 0 0

The value must lie within the bounding box of the dataset. It will default to the min,min,min in each dimension.

Resolution Set the sampling resolution for the generated volume. 100 100 100
Scale Set scale factor for a unit volume. 1 1 1
