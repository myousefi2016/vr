PacMan (PacMan)

Property Description Default(s) Restrictions
Center This property specifies the 3D coordinates for the center of the sphere. 0.0 0.0 0.0
ThetaResolution The value of this property represents the number of divisions between Start Theta and End Theta around the sphere. (See the Start Theta and End Theta properties.) The theta divisions are similar to longitude lines on the earth. The higher the resolution, the closer the approximation will come to a sphere, and the more polygons there will be. 60
StartTheta To form a complete sphere, the value of this property should be 0 degrees, and the value of the End Theta property should be 360 degrees. The value of this property can be adjusted to form only a portion of a sphere. 30
EndTheta The value of this property can be adjusted to form only a portion of a sphere. This value is measured in degrees. 330
PhiResolution The value of this property represents the number of divisions between Start Phi and End Phi on the sphere. (See the Start Phi and End Phi properties.) The phi divisions are similar to latitude lines on the earth. 60
