Moment Vectors (MomentVectors)

Convert scalars to flux or circulation vectors.
Often times a simulation that computes circulation or flux in a mesh will simply write out the scalar that represents the amount of movement and the direction is implied by the normal of the surface cell (for flux) or the direction of the line cell (for circulation). The array values associated with the 2D and 1D geometric elements will be converted to a vector pointing in the direction of the flow.
Property Description Default(s) Restrictions
Input The input to the filter.

Accepts input of following types:

  • vtkDataSet

The dataset must contain a field array (cell) with 1 component(s).

Flux/Circulation Array The name of the scalar array that represents flux through a 2D element or circulation along a 1D element. This array will be converted to a vector pointing in the direction the flow or circulation is moving.

An array of scalars is required.

The value must be field array name.
Input Moment Is Density If off, then the input array is taken to be the total flux through or circulation along each element. If on, then the input array is taken to be the density of the flux or circulation. 0

Accepts boolean values (0 or 1).
