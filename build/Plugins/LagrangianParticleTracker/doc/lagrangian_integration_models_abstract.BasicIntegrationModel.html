Locator Specify the locator used to locate cells in the flow intput

The value can be one of the following:

  • CellLocator (cell_locators)
  • CellTreeLocator (cell_locators)
  • ModifiedBSPTree (cell_locators)

NonPlanarQuadSupport If NonPlanarQuadSupport is true, surface interaction for vtkQuad will use the bilinear patch code to compute intersection. 1

Accepts boolean values (0 or 1).
Particle Initial Velocity This property contains the name of the vector array from the seeds to use as initial particle velocity. 0;1;0;0;ParticleVelocity

An array of vectors is required.

The value must be field array name.
UseInitialIntegrationTime Set whether to use an initial integration time array from input instead of zero values. 0

Accepts boolean values (0 or 1).
Particle Initial Integration Time This property contains the name of the scalar array from the seeds to use as initial particle integration time. 1;1;0;0;ParticleInitialIntegrationTime

An array of vectors is required.

The value must be field array name.
