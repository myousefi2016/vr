Property Description Default(s) Restrictions
Flow Velocity This property contains the name of the array to use as flow velocity. 3;0;0;3;FlowVelocity

An array of vectors is required.

The value must be field array name.
Flow Density This property contains the name of the array to use as flow density. 4;0;0;3;FlowDensity

An array of scalars is required.

The value must be field array name.
Flow Dynamic Viscosity This property contains the name of the array to use as flow dynamic viscosity. 5;0;0;3;FlowDynamicViscosity

The value must be field array name.
Particle Diameter This property contains the name of the array to use from seeds as particle diameter. 6;1;0;0;ParticleDiameter

The value must be field array name.
