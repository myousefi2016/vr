Geodesic Measurement (GeodesicMeasurement)

Property Description Default(s) Restrictions
Input

Accepts input of following types:

  • vtkPolyData
Endpoints The input that provides the endpoints between which geodesic paths will be computed.

Accepts input of following types:

  • vtkPointSet
Create Loop If on, enables loop creation by computing a path from the last point to first point in the endpoints. 0

Accepts boolean values (0 or 1).
Create Loop with Line If on and if Create Loop is on, uses a straight line segment to connect the last point to the first point in the endpoints. 0

Accepts boolean values (0 or 1).
