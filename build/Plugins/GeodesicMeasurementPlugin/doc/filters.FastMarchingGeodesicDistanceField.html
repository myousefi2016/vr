Fast-Marching Geodesic Distance-Field From Binary Field (FastMarchingGeodesicDistanceField)

Compute the distance of points of the mesh to chosen seeds
Compute the distance of each point of the mesh to a set of seeds defined as points with non-zero value. Internal Fast-Marching-Method plugin is used for computational efficiency.
