GMV Reader (GMVReader)

Read binary or ASCII files stored in GMV format.
The GMV reader reads binary or ASCII files stored in GMV format. The default file extension is .gmv. The output of this reader is a multi-block dataset.
Property Description Default(s) Restrictions
FileName This property specifies the file name for the GMV reader.

The value(s) must be a filename (or filenames).
PointArrayInfo
Point Arrays This property contains a list of the point-centered arrays to read.

The list of array names is provided by the reader.
CellArrayInfo
Cell Arrays This property contains a list of the cell-centered arrays to read.

The list of array names is provided by the reader.
FieldArrayInfo
Field Arrays This property contains a list of the global variables to read.

The list of array names is provided by the reader.
HasTracers
ImportTracers If this property is set to 1, tracers will be read (and stored in a separate block). 1

Accepts boolean values (0 or 1).
HasPolygons
ImportPolygons If this property is set to 1, polygons will be read (and stored in a separate block). 1

Accepts boolean values (0 or 1).
