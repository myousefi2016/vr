GMV Reader (GMVSeriesReader)

Read binary or ASCII files stored in GMV format.
The GMV reader reads binary or ASCII files stored in GMV format. The default file extension is .gmv. The output of this reader is a multi-block dataset. This reader also supports file series.
Property Description Default(s) Restrictions
FileNameInfo
FileNames The list of files to be read by the reader. If more than 1 file is specified, the reader will switch to file series mode in which it will pretend that it can support time and provide 1 file per time step.

The value(s) must be a filename (or filenames).
TimestepValues Available time step values.
Ignore stored time step values / PROBTIME keyword Toggle whether to ignore time step values stored with PROBTIME keyword. If unchecked, the entire file sequence (!) is parsed once to determine time step values. Depending on its total size, this can take a considerable amount of time. Also, if checked, files with duplicate time step values will be ignored. (Convenient for restarted simulations where simulated times overlap.) If checked, fake time step values will be used. This allows for stepping through all files of a given series even if all files contain the PROBTIME keyword and all specify a dummy setting of e.g. 0.000. 1

Accepts boolean values (0 or 1).
